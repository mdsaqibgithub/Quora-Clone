# 🧠 Quora Clone – TransportSimple Assignment

This is a Django-based web application inspired by Quora, built as part of the Django Developer assignment for **TransportSimple**.

---

## ✅ Features Implemented

- 🔐 User Registration & Login (Django built-in auth)
- 📝 Post Questions
- 👀 View All Questions
- 💬 Answer Questions
- 👍 Like Answers
- 🚪 Logout

---

## ⚙️ Tech Stack

- **Backend**: Django (Python)
- **Frontend**: HTML&CSS (basic templates with Django forms)
- **Database**: SQLite (default with Django)
- **Auth**: Django's built-in authentication
